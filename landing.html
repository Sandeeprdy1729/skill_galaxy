Connect Claude Desktop Directly

Install the SkillGalaxy MCP server and let Claude Desktop search, browse, and download skills without leaving the conversation. Real-time access to the full catalog.

Setup Guide → ⭐ Star on GitHub 
// claude_desktop_config.json
{
  "mcpServers": {
    "skillgalaxy": {
      "command": "node",
      "args": ["path/to/mcp-server/index.js"],
      "env": {
        "SUPABASE_URL": "your-url",
        "SUPABASE_KEY": "your-key"
      }
    }
  }
}
