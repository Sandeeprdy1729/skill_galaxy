Connect Claude Desktop
Add SkillGalaxy MCP to Claude Desktop
One config change gives Claude direct access to all 9,872+ skills — no manual downloads.
1. Install the MCP server
cd /path/to/skill_galaxy/mcp-server npm install npm run build-skills
2. Add to your Claude Desktop config
📁 Config file location: macOS: ~/Library/Application Support/Claude/claude_desktop_config.json  ·  Windows: %APPDATA%\Claude\claude_desktop_config.json
{ "mcpServers": { "skillgalaxy": { "command": "node", "args": ["/FULL/PATH/TO/skill_galaxy/mcp-server/index.js"] } } }
3. Restart Claude Desktop
Claude can now search_skills, get_skill, list_categories, and retrieve any of the 9,872+ skills on demand — without manual downloads.
💡 Tip: Replace /FULL/PATH/TO/skill_galaxy with the actual absolute path where you cloned this repo.
